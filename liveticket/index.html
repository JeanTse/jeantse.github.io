チケット購入ガイド（海外向け）

中国コンサートチケット販売における「強実名制」実施要項：
厳格な本人確認：チケット購入時・入場時に、記入された購入者情報と本人確認書類（中国国民身分証/外国籍パスポート）の完全一致が必須。
ダフ屋対策の徹底：本制度の目的はチケット転売屋（ダフ屋）の排除にあり、マーズ・コンサートでは公式価格を超える高額チケットの流通が物理的に不可能！
詐欺注意喚起：公式価格を上回る金額でのチケット転売を謳う者は全て詐欺師だ。被害防止のため絶対に取引しないでください。

  • APKファイルをダウンロードしてインストール：「Trip.com」アプリ、「大麦」アプリ、「猫眼」アプリ

  • WeChatユーザーの方は、自動翻訳機能を内蔵したWeChatミニプログラムがお勧め

チケット購入‌

通常2-3回のチケット販売ラウンドが実施される。初回販売では全チケットが販売され、第2ラウンドで払い戻しチケットを再販売。ステージ設置後に新たな観覧エリアが追加された場合に限り第3ラウンドが行われる。第2・3ラウンドのチケット数は少数となる

「Trip.com」アプリ

  • 「アカウント → 旅行者・連絡先 → 保存済みのお客様情報 → 新規旅行者を追加」ページで、購入者の（パスポート情報と一致の）身分証明情報を事前記入

  • 事前記入の購入者情報の人数は、チケット購入数量と一致するのを確保

  • 「アカウント → 保存済みのカード情報 → カードを登録」ページで、クレジットカード情報を事前記入

  • アプリのホームページで「Hua Chenyu コンサート」を検索し、希望公演の詳細ページに進む

  • 正しい公演日時と希望購入価格を選択し、「発売時にリマインド」をクリック（「カレンダーに追加」をクリックしてアラームを設定することも推奨）

  • カウントダウンに従い、最後の1秒で速やかにクリックして確認ページに進む。購入情報が正しければ「支払う」をクリックする

  • 支払い方法（クレジットカード、AliPay（アリペイ）、WeChat Pay（微信支付））を選択する

「大麦」と「猫眼」アプリ

  • アプリのホームページで「华晨宇」を検索し、希望公演の詳細ページに進む

  • 詳細ページの真ん中には、いつも「预填信息（事前情報登録）」のボタンがあり、ここで購入者の（パスポート情報と一致の）身分証明情報と携帯番号を事前記入のが推賞

  • 事前記入の購入者情報の人数は、チケット購入数量と一致するのを確保

  • 正しい公演日時、希望購入価格とチケット購入数量を選択し、「预约抢票（チケットを予約）」をクリックし、販売開始のタイムを待つ

  • チケット販売開始のカウントダウンが0になった瞬間に購入ボタンをクリック、支払い画面へ遷移するまで（即ち、注文が受領されるまで）リフレッシュを繰り返す

注文決済

支払い時間順に座席が割り当てられるため、優良席を確保するには速やかな決済が必須

「Trip.com」アプリ

  • クレジットカード（UnionPay、VISA、Master、AMEX、JCB、DISCOVER）、ApplePay、Paypalなど

「大麦」アプリ

  • AliPay（アリペイ） と WeChat Pay（微信支付）

「猫眼」アプリ

  • WeChat Pay（微信支付）のみ

入場‌

  • 「人工通道（有人ゲート）」を利用必須（自動改札機は中国身分証のみ対応）

  • 購入アカウントにログインし、「票夹（電子チケット管理）」または「Trip.com → 注文 → イベントチケット」から該当注文のQRコードを表示

  • 購入時に使用したパスポート原本およびQRコードをスタッフに提示

払い戻し

  • 同一注文内の複数チケットは一括返金必須（単独返金不可）

  • 初回販売後24時間以内は無条件・無料返金可能

  • 24時間経過後は手数料20%～50%が発生（公演日が近づくほど手数料率上昇/航空券返金規則に類似）

  • 公演開始10日前から返金不可

  • 疾病や不可抗力事由（天災・戦争等）による観覧不能の場合、サイトカスタマーサービスに連絡し審査通過後に無料返金可能

まとめとアドバイス

  • パスポートの氏名・番号を‌正確に‌入力すること
  • 入場時は必ずパスポート‌原本‌を携帯（コピー不可）
  • 複数の友人を動員し、各アプリで複数アカウントを登録。分散配置で同時にチケット争奪作戦を実行すれば、入手成功率が格段に向上する
